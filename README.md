# 抖音WebApp(React + typescript)
* 单应用抖音WebApp,目前该版本还会在不断完善

### 启动服务
> 运行本项目前，请先开启mockServer
```
  npm install      // 下载依赖
  npm run mock     // 运行mock服务
  npm run dev      // 运行项目
```
